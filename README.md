# CV Retro

Pagina web personale in stile retro anni '90, pensata come CV interattivo.

## Contenuto
- Sezioni principali: About Me, Experience, Education e Intern.
- Sidebar con icone e popup informativi.
- Layout responsive per desktop e mobile.

## Struttura
- `index.html`: struttura della pagina e logica popup.
- `style.css`: stile grafico retro.
- `static/`: immagini e loghi.

## Avvio rapido
Apri `index.html` nel browser oppure avvia un server statico locale dalla cartella del progetto.
